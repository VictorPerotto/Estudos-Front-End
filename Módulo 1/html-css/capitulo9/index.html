Listas

Listas ordenadas

Podemos criar listas ordenadas, ou seja, que seguem uma ordem especifica, como uma receita de bolo, usando a tag OL de ordered list e depois para cada item da lista usamos a tag LI

  1. Farinha
  2. Ovo
  3. Leite
  4. Chocolate em pó

Listas desordenadas

As listas desordenadas como o nome já diz, são aquelas que não seguem uma ordem especifica, podemos criar uma lista desordenada através da tag UL de unordered list, e para cada item novamente usamos a tag LI

  • Tirar o lixo
  • Lavar a louça
  • Levar o cachorro para passear
  • Fazer o almoço
