Semântica

Negrito / Destaque

Podemos colocar algo em negrito usando a tag B (não semântica e não recomendada)

Podemos colocar algo em destaque usando a tag Strong (forma semântica)

Itálico / Ênfase

Podemos colocar algo em itálico usando a tag I (não semântica e não recomendada)

Podemos colocar algo em ênfase usando a tag EM (forma semântica)

Texto marcado

Podemos marcar um termo em um texto usando a tag Mark

Podemos marcar em vários textos

Podemos utilizar uma estilização unica inline ao invés de geral

Texto grande e pequeno

Podemos também usar uma tag para texto grande B(não semantica)

E uma para textos pequenos Small

Texto deletado e inserido

Podemos colocar um texto e marcar ele como deletado usando a tag Del

E da mesma forma podemos marcar um texto como inserido usando a tag Ins

Texto sobrescrito e subscrito

Podemos colocar um texto como x20+3 usando a tag Sup

Ou podemos colocar algo como H2O usando a tag Sub
